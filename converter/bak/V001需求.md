# 规则转换工具 (Rule-Converter) 完整需求文档

## 1. 基本信息
- **版本**: V0.01
- **更新日期**: 2026-01-31
- **目标平台**: Python 3.14 (兼容 3.10+)

## 2. 处理任务定义
程序需同时处理两组 GitHub 资源：
1. **Gemini 组**: 
   - 源: https://raw.githubusercontent.com/Hexon-X/Hex-Clash/refs/heads/main/list/Ai/Gemini.list
   - 输出: Gemini.cvc, Gemini.list
2. **OpenAI 组**: 
   - 源: https://raw.githubusercontent.com/Hexon-X/Hex-Clash/refs/heads/main/list/Ai/OpenAI.list
   - 输出: OpenAI.cvc, OpenAI.list

## 3. 业务逻辑规范
### 3.1 镜像备份 (.list 文件)
- **处理逻辑**: 纯镜像下载。
- **要求**: 原始文件的内容、注释、空行、顺序必须 100% 保留。

### 3.2 规则转换 (.cvc 文件)
- **文件头**: 固定首行为 `payload:`。
- **过滤机制**: 
  - 丢弃所有以 `#` 开头的行。
  - 丢弃所有空白行。
  - 仅保留前缀为 `DOMAIN,`、`DOMAIN-KEYWORD,`、`DOMAIN-SUFFIX,` 的行。
- **格式转换**: 将有效行修改为 `  - 内容` 格式（2空格缩进）。
- **去重逻辑**: 基于最终转换后的行进行去重，且必须保留原始出现的先后顺序。

## 4. 存储规范
- **存储目录**: 固定为 `d:\tmp`。
- **编码格式**: 全局使用 `UTF-8`。
